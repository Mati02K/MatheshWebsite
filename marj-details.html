In my last year of school(12th std) I had done a Governmental E-Helpline application where the users can pay their electricity bill, apply for smart card, calculate their Income Tax, download the government forms, get the government helpline numbers all in a single application. The application is created using JAVA and the database used for storing here is MYSQL(local mysql in my computer). The connection to the Java and the MYSQL is done through the JDBC driver.
